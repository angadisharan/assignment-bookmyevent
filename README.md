# BookMyEvent (modular monolith)

Kotlin + Spring Boot 3.x sample application scaffold for the BookMyEvent project.

Prereqs:
- JDK 17
- Maven
- MySQL (or configure datasource in `application.yml`)

Run:

```bash
mvn spring-boot:run
```

Notes:
- Project uses Kotlin and Spring Boot 3.x.
- Update `src/main/resources/application.yml` with your MySQL credentials.
